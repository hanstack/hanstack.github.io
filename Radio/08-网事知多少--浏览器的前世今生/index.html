    08-网事知多少--浏览器的前世今生¶

    08-网事知多少--浏览器的前世今生

    20世纪90年代开始，出现了有史以来最具影响力的发明之一；万维网。设在瑞士日内瓦的欧洲核子研究中心（CERN）是一个由欧洲多国政府资助的高性能物理实验室，同时也是欧洲最大的物联网节点。这里的一名英国工程师蒂姆·伯纳斯·李（Tim Berners-Lee）意识到，将超文本模式应用到互联网上，就可能创造出一个全球性的网络，这是原来超文本的发明者不曾想到过的。他着手定义一种HTML.(超文本标记语言)用来书写超文本文件，以便将他们连接在一起。1945年 1945年，万尼瓦尔·布什（Vannevar Bush）提出了被称为“存储扩充器”（memory extender）的概念，这是具有开创性的信息组织方法，形成了今天超文本的基础。文本间的交叉引用链接使使用者可以用非线型的方式查阅信息，类似于人类思维的方式——也就是说，通过联想而不是直线顺序。为了制造这些“联想”链接，单个的因素会被标上标签，从而可以得到打开系统的动态链接。设计师的任务是创建内容，决定链接，用非线型组织信息的方式做出简易而明确的引导。

    1963年，德特·纳尔逊(Ted Nelson)创造了术语“超文本”。1981年，德特在他的著作中使用术语“超文本”描述了这一想法：创建一个全球化的大文档，文档的各个部分分布在不同的服务器中。通过激活称为链接的超文本项目，例如研究论文里的参考书目，就可以跳转到引用的论文。

    超文本是计算机出现后的产物，它以计算机所储存的大量数据为基础，使得原先的线性文本变成可以通向四面八方的非线性文本，读者可以在任何一个关节点上停下来，进入另一重文本，然后再点击、进入又一重文本，理论上，这个过程是无穷无尽的。从而，原先的单一的文本变成了无限延伸、扩展的超级文本、立体文本。

    这样一种新鲜事物的出现显然和技术的发展息息相关。电子媒介的崛起深刻改变了世界的文化面貌。电影、电视可以把纸面上的文学转换成可视可听的电子形式，计算机的技术条件所提供的“超文本”使罗兰·巴特设想过的“可写文本”变成了现实

    伯纳斯·李在一台NeXT电脑上实现了服务器的功能，并且写出了第一个客户端软件，也就是他称之为“万维网”的浏览器。服务器能够根据一个简单的协议(超文本传输协议)HTTP将网页传送到客户端。这个浏览器受到了“带脑子图书技术公司”（Electronic Book Technologies）在1990年开发的DynaText软件工具的很大启发。

    ​ 比较客观的来说HTML只是发布文件的一种语言而已，在此之前也是非常非常的常见，HTML编码主要是告知浏览器怎么样去显示页面内容。但是HTML比较创新的是“链接”功能，用户能够使用鼠标点击“ 链接”就能够跳转到另一个页面。这种方式大家应该并不是很陌生，当我们打开百度搜索的时候，你搜索出来的结果前三个一定是相关的广告的超链接，此时你只需要用鼠标点击超链接就能愉快的进入广告页面了。超文本传输协议使得我们访问的页面可以在同一台服务器上也可以在不同的服务器上。这个创意改变了用户在互联网上的体验。

    1990年，在蒂姆伯纳斯·李博士在写出了世界上第一个浏览器客户端程序之后的同年12月，CERN首次启动了万维网并成立了全球第一个WWW网站info.cern.ch。这个网站至今仍然可以访问。并且如果你真的去访问这个网站的话，你会发现网站其实是非常非常简单的没有任何花里胡哨的东西，这个网站至今仍然是CERN的官方网站。有兴趣的话可以点进去看看。

    ​ 1991年12月，物理学家保罗·昆茨（paul·Kunz）在斯坦福直线加速中心（SLAC）建立了美国的第一个互联网服务器。赫尔辛基理工大学的4名学生Kim Nyberg、Kari Sydänmaanlakka、Teemu Rantanen和Kati Suominen为X Window（运行X的Unix）共同开发了一种新颖的图形浏览器，他们为Erwise加入了如文本搜索、载入多个Web页面等功能，这些功能要在很久之后才能在其它浏览器上看到。美国的第一个主要的浏览器是violaWWW，是中国台湾出生的学生魏裴元在1992年12月的加州大学伯克利分校完成的，这个浏览器是魏裴元用自己的自创的语言开发的，语言名叫做viola也就是中提琴的意思，这真的是一个非常有艺术气质的程序员哈哈哈。这是计算机历史上第二个图形化的浏览器。

    1993年3月2日中国科学院高能物理研究所租用AT&T公司的国际卫星信道建立的接入美国SLAC国家实验室的64K专线正式开通，成为我国部分连入Internet的第一根专线。这根专线在1994年4月中国正式连入Internet后，也实现了Internet的全线连通。”

    美国伊利诺伊大学的马克·安德里森（Marc Andreessen）和埃里克·比那（eric Bina），创造出了一个带有图形界面易于使用的浏览器。1993年，他们完成了Mosaic网站浏览器。

    Mosaic的图形用户界面带来了重大变化，能够非常方便的显示文本和图像。它原本只是运行在Unix系统上，但是很快便移植到windows操作系统上，它把每一台电脑都变成了互联网的客户端。

    1994年4月，马克·安德森（Marc Andreessen）和吉姆·克拉克（Jim Clark）成立了 Mosaic Communications Corporation。Mosaic 是一个软件的名字，它允许用户访问网络上的不同内容。安德森在伊利诺伊大学的国家超级计算应用中心（NCSA）工作时，曾参与过这个项目。

    克拉克此前曾在 Silicon Graphics 工作，他带着几位同事一起参与 Mosaic 的工作。Andreessen 还从 NCSA 挖来了几名前同事。到1994年10月，该团队已经发布了Mosaic Netscape 0.9。12月，他们将公司改名为 Netscape Communications ，并推出了 Netscape Navigator 1.0版本，也就是我们大家熟悉的Netscape 。

    商业用户应该以每用户99美元的价格购买该浏览器的许可证，其中包括保修和客户支持，但这并没有持续多久。同样，你也可以在零售店中找到盒装版的 Netscape，在某些时候每份售价 40 美元。

    出于所有实际目的，在当时，Navigator 是唯一公开发行的网络浏览器，因此它享受了一段几乎零竞争的时期。

    1995年8月9日，Netscape 公司上市，其IPO的售价为每股28美元。该股原本应该以每股14美元的价格发行，但在最后一刻决定将价格翻倍。在首日交易中，该股涨至每股75美元，市值达到30亿美元，首日涨幅惊人。Netscape 的 IPO 掀起了创投对互联网公司的广泛投资，也在后来创造了网络泡沫。

    那是一个神奇的时代，家用电脑的销售刚刚蓬勃发展，如果你足够幸运，你的个人电脑会配备一个拨号上网的调制解调器。你会听到电话线连接世界的争吵声。启动 Netscape，盯着跳动的动画，同时加载一个网页。

    然而，谁也不知道，微软一直在研发自己的浏览器。就在 Netscape 公司首次公开募股的几天后，它发布了 Windows 95 和 Internet Explorer 1.0。在接下来的一年时间里，两家公司展开了激烈的竞争，微软总是落后一步。

    直到1996年8月Internet Explorer 3.0发布。微软终于在浏览器技术上追上了 Netscape。逐渐的，微软通过在每个 Windows 系统中预装 IE 浏览器来抢夺市场份额。

    Netscape 公司继续致力于 Navigator 浏览器和 Communicator 的开发，尽管捆绑和名称的改变一直让用户感到困惑。1998年初，该公司宣布计划开源 Communicator 源代码，这也促使了 Mozilla 项目的成立，这是一项开放源码的工作，后来成为 Firefox。

    Netscape 在发布源代码之后，浏览器的更新速度就开始放缓。而微软则马不停蹄地进行更新。微软已经赢得了大部分的市场。这种浏览器偏好的转变标志着 Navigator 浏览器漫长的螺旋式死亡开端。

    而伴随着 IE 5.0 的发布，微软在浏览器上的优势已经明显超越 Navigator 浏览器，网站的图形化程度越来越高，网速越来越快，相比较而言，Navigator 浏览器的错误更多、速度更慢，而且更容易崩溃。

    根据当时的记录，在 90 年代末，微软每年花费 1 亿多美元开发 IE 浏览器，开发团队超过 1000 人。

    到 1998 年，昔日的互联网之王陷入困境。AOL 看到了这款浏览器的潜力，在1998年11月以 42 亿美元的价格进行收购。

    Navigator/Communicator浏览器（现在已被称为 Netscape）的开发工作进展缓慢。即使在Mozilla 项目的帮助下，AOL 直到 2000 年才发布 Netscape 6，使其在浏览器大战中远远落后。又有两年时间，浏览器将在最后的死亡边缘挣扎。

    2002年8月，Netscape 7发布并开始了漫长的告别。第二年，AOL 宣布关闭 Netscape 部门，并裁掉了大部分员工。基于 Firefox 源代码，虽然开发工作又持续了几年时间，但在这个品牌下并没有发布什么重要的产品。

    2005 年，AOL 将开发职责交给了一家名为 Mercurial Communications 的外部加拿大公司。Mercurial 在2005年5月发布了 “Netscape Browser 8”。在接下来的两年里发生了几次迭代，最后一次更新是2007年4月。

    就像自己的孩子不忍心放弃一样，AOL 又开始了内部开发。它将浏览器重新命名为 Netscape Navigator，并在2007年10月推出了第9版。随后，AOL 只继续支持了几个月，就终于肯放手了。
