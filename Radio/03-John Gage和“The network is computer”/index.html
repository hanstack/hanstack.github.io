    03-The network is computer

    大家好，欢迎大家收听半吊子工程师电台。这是我的第三期电台了，这期电台我打算谈点新的东西，打算从我能找的资料聊一个人John Gage.John Gage是sun公司的第21名员工，也是sun公司的联合创始人之一也是该公司的首席研究员。在这里顺便说一下，可能大家对sun公司应该有简单的了解，因为在国内无论是教学还是能够找到的资料上，我看到大部分的描述或者是媒体的文章上都是把sun称之为太阳公司，这个其实是有点问题的。虽然说sun这三个字母组合起来是太阳的意思，国内的话也直接发音为sun，但是实际上sun是三个英文首字母的缩写是Stanford University Network的缩写，我们所称之为的sun公司或者太阳公司的全程应该是，Stanford University network Mirrorsystem公司。也就是斯坦佛大学网络微系统公司。他是由斯坦佛大学的校园的几个校友创办的。当年的sun公司在硅谷也是一时风头无量，和微软，思科，惠普有相同的江湖地位，sun公司在成立之初就提出了开放标准，互操作性，异构和参与等思想，希望能通过开放的额社区讲世界各地的人们联系在一起，这个在当时的环境中是非常非常激进的思想，但是多年之后，这些思想在世界各地开花结果。虽然最后我们没有看到sun公司与这个时代互联网巨头并肩称雄，但是sun公司依旧在很多方面不断的影响着我们，比如sun公司推出的编程语言java和sun公司的John Gage提出的“The network is the computer”也一直在影响着这个时代的技术的发展。java作为一门语言已经走过了几十年的时间，在这几十年的岁月里，不断有新的语言出现了，不断有新的语言没落了，可是java依旧是老当益壮，这是sun留给这个世界最宝贵的资产之一。当然今天在这里我们不谈java，因为我对编程语言并不是非常的了解，也不能讲好java所以我今天主要是，“the network is the computer”来讲。如果大家对java有兴趣的话可以听一下软件那些事儿的电台，谁实话，我做这个电台很大程度是收到这个电台主播的影响，当然自己也确实比较闲。做电台的一个好处就是，当你确定了一个主题之后你会有意识的去收集相关的资料，这个过程我个人觉得还是非常非常的有趣。

    说了这么多现在开始步入正题，这一期的主题是“teh network is the computer”。这是sun公司的员工John Gage所提出的一个口号。John Gage出生在洛杉矶，在六十年代他是加州伯克利大学的一名大学生。他第一次遇到了Biil Joy。并解决了Biil Joy提出的一个关于排版数学方式的问题。在1982年，John Gage再次遇到了Biil Joy。Biil Joy在餐巾纸上画了一张草图，有点类似于IBM的6100 工作站，但是能够运行unix系统并且使用TCP/IP链接，使用斯坦福大学的工程师设计的微处理器上运行。这启发了John Gage，他马上意识到，这是一个潜在的突破性的产品；可以在网络上运行的廉价的功能强大的计算机。随后两人联合其他加入的12个合作伙伴共同创立的sun mirrorsystem公司。

    “teh network is the computer”网络就是计算机这个词是由john Gage 在1982年提出的，是数十年 Sun MicroSystems的口号。事实上，从1982年提出一来，已经过去了四十年了。在今天蓬勃发展的云计算算承载了大量的计算资源，我们也能够通过网络很轻松的链接到这些计算资源，从用户的角度来看，这些放置在远端的资源可能比放置到本地更加有效。也许服务器乘坐服务器的计算节点实际上知识网络计算机的组件，当然我说这句话是根据John Gage说的网络就是计算机来说的。IBM 3270 是1983年发布的，这是一台支持多硬件模拟的终端设备，可以当做大型机的终端来使用，这就有点像如今我们使用终端软件或者直接在网页上通过网络的方式连接到云服务提供商提供的弹性云服务器上，国外的AWS，微软，国内的阿里云，腾讯云，华为云等。

    如今我们已经实现了云计算;云计算一般称之为cloudcomputing 其中的cloud实际上指的是network的意思，云计算也就是通过网络连接计算资源的意思，cloud也就是network。也许是在“the network is the computer”的指导下成就了现在的云计算；也许是“the network is the computer”预言了云计算的发展；但是无论如何，当我们在使用云计算带来的便利时，我们也许应该知道曾经有一个公司，他提出了“the network is the computer的口号”；当然这并不能给我们这些做实施的工程师带来什么。我们的工作接触到的还是黑的方的硬的服务器，还死沉。

    如今的云计算已经能够突破地域的限制，我们可以通过网络调度世界各地的计算资源。也许经历过更长时间的迭代，云计算也许会变成星际的；当时这个前提是我们需要解决网络的问题，比如时延。也许这也是为什么云计算的云在计算前面，可能是网络比较重要吧。哈哈 这是我瞎猜的。

    John Gage 还做过一个叫做netday的项目；这个是在1995年成立的一项活动，呼吁高科技公司向全球的学校，图书馆和诊所投入资源，帮助他们可以连接到internet。当时sun公司的市值已经到达60亿美元，在我们的理解中一个60亿美元资产的企业的高级领导者会选择搞这个NetDay这样的项目把自己搞的很累呢；这种事情费力不讨好，因为是个非盈利项目也不挣钱。当时一部分的原因可能是为了展示sun的技术力量，当时sun公司的硬件站到全球web服务器的三分之一以上。sun公司有1500名员工，每天产生多达200万封电子邮件。

    当时他们对NetDay提出的愿景是是通过电子邮件和web才能使用netday,他也没有办公室，没有电话号码，没有传真号码。没有带薪员工。这是一个虚拟的公司，或者说这是一个完全在互联网上的公司。真的前卫又牛逼啊。

    想想现在我们还在公司做社畜，996是吧。哈哈哈
